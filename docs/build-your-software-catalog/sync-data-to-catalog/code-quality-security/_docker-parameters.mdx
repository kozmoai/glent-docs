| Parameter                                           | Description                                                                                                                                                                                                                                                                              | Required |
| --------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| `OCEAN__INTEGRATION__CONFIG__SONAR_API_TOKEN`       | The [SonarQube API token](https://docs.sonarsource.com/sonarqube/9.8/user-guide/user-account/generating-and-using-tokens/#generating-a-token)                                                                                                                                            | ✅       |
| `OCEAN__INTEGRATION__CONFIG__SONAR_ORGANIZATION_ID` | The SonarQube [organization Key](https://docs.sonarsource.com/sonarcloud/appendices/project-information/#project-and-organization-keys) (Not required when using on-prem sonarqube instance)                                                                                             | ✅       |
| `OCEAN__INTEGRATION__CONFIG__SONAR_IS_ON_PREMISE`   | A boolean value indicating whether the SonarQube instance is on-premise. The default value is false                                                                                                                                                                                      | ✅       |
| `OCEAN__INTEGRATION__CONFIG__SONAR_URL`             | Required if using **On-Prem**, Your SonarQube instance URL                                                                                                                                                                                                                               | ❌       |
| `OCEAN__GLINT__CLIENT_ID`                            | Your glint client id ([How to get the credentials](https://docs.useglint.io/build-your-software-catalog/custom-integration/api/#find-your-glint-credentials))                                                                                                                             | ✅       |
| `OCEAN__GLINT__CLIENT_SECRET`                        | Your glint client secret ([How to get the credentials](https://docs.useglint.io/build-your-software-catalog/custom-integration/api/#find-your-glint-credentials))                                                                                                                         | ✅       |
| `OCEAN__INITIALIZE_GLINT_RESOURCES`                  | Default true, When set to true the integration will create default blueprints and the glint App config Mapping. Read more about [initializeGlintResources](https://ocean.useglint.io/develop-an-integration/integration-configuration/#initializeportresources---initialize-glint-resources) | ❌       |
| `OCEAN__INTEGRATION__IDENTIFIER`                    | The identifier of the integration that will be installed                                                                                                                                                                                                                                 | ✅       |
