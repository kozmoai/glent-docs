**Advanced configuration**

| Parameter                        | Description                                                                                                                                                                                                                                                                                    |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `integration.eventListener.type` | The event listener type. Read more about [event listeners](https://ocean.useglint.io/framework/features/event-listener)                                                                                                                                                                         |
| `integration.type`               | The integration to be installed                                                                                                                                                                                                                                                                |
| `scheduledResyncInterval`        | The number of minutes between each resync. When not set the integration will resync for each event listener resync event. Read more about [scheduledResyncInterval](https://ocean.useglint.io/develop-an-integration/integration-configuration/#scheduledresyncinterval---run-scheduled-resync) |
| `initializeGlintResources`        | Default true, When set to true the integration will create default blueprints and the glint App config Mapping. Read more about [initializeGlintResources](https://ocean.useglint.io/develop-an-integration/integration-configuration/#initializeportresources---initialize-glint-resources)       |
