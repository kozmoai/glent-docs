| Parameter                                   | Description                                                                                                                                                      | Required |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| `OCEAN__INTEGRATION__CONFIG__OPENCOST_HOST` | The Opencost server URL                                                                                                                                          | ✅       |
| `OCEAN__INITIALIZE_GLINT_RESOURCES`          | Default true, When set to false the integration will not create default blueprints and the glint App config Mapping                                               | ❌       |
| `OCEAN__INTEGRATION__IDENTIFIER`            | Change the identifier to describe your integration, if not set will use the default one                                                                          | ❌       |
| `OCEAN__GLINT__CLIENT_ID`                    | Your glint client id ([How to get the credentials](https://docs.useglint.io/build-your-software-catalog/custom-integration/api/#find-your-glint-credentials))     | ✅       |
| `OCEAN__GLINT__CLIENT_SECRET`                | Your glint client secret ([How to get the credentials](https://docs.useglint.io/build-your-software-catalog/custom-integration/api/#find-your-glint-credentials)) | ✅       |
| `OCEAN__GLINT__BASE_URL`                     | Your glint base url, relevant only if not using the default glint app                                                                                              | ❌       |
