:::note Prerequisites
To install and use Glint's Pulumi provider, you will need to install the [Pulumi CLI](https://www.pulumi.com/docs/get-started/install/)
:::

To install the Pulumi provider, use your preferred method of installing Pulumi providers.

## Configuration

Please make sure to configure the clientId and secret to Glint using the commands

<FindCredentials />

```bash showLineNumbers
pulumi config set glint:clientId <clientId>
pulumi config set glint:secret <clientSecret> --secret
```
